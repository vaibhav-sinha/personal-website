The **Markov property** says the future depends only on the current state and action, not on the full history. Formally, for any history <Math latex="h_t = (s_0, a_0, r_1, s_1, \ldots, s_t)" />:

<Math
  display
  latex="P(s_{t+1} \mid h_t, a_t) = P(s_{t+1} \mid s_t, a_t)"
/>

This assumption is what makes MDPs tractable. Without it, the agent would need to remember the entire trajectory to make good decisions.

In practice, you often **engineer state representations** so the Markov property holds — for example, including velocity in a robot's state, not just position.
