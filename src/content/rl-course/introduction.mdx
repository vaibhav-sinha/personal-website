Reinforcement learning (RL) is about learning what to do — how to map situations to actions — so as to maximize a cumulative reward. Unlike supervised learning, an RL agent is not told which actions are correct. It must discover them through trial and error.

## The basic loop

At each timestep, an agent observes the state of its environment, takes an action, and receives a reward signal. The goal is to learn a policy that maximizes expected cumulative reward over time.

<CodeBlock
  lang="python"
  title="agent_loop.py"
  code={`
def agent_loop(env, policy):
    state = env.reset()
    total_reward = 0

    while not env.done:
        action = policy(state)
        state, reward, done = env.step(action)
        total_reward += reward

    return total_reward
`}
/>
