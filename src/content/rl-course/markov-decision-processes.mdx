A **Markov decision process** (MDP) is the standard mathematical framework for modeling sequential decision problems. An MDP is defined by a tuple \((S, A, P, R, \gamma)\):

- **S** — set of states
- **A** — set of actions
- **P** — transition dynamics \(P(s' \mid s, a)\)
- **R** — reward function \(R(s, a, s')\)
- **γ** — discount factor in \([0, 1]\)

See [The Markov Property](/rl-course/markov-decision-processes/the-markov-property) for why this tuple is enough to describe a sequential decision problem without tracking full history.

## Looking ahead

With MDPs defined, the next step is to quantify how good it is to be in a state or to take an action — the value functions <Math latex="V(s)" /> and <Math latex="Q(s, a)" />.

The state-value function under policy <Math latex="\pi" /> is the expected discounted return:

<Math
  display
  latex="V^\pi(s) = \mathbb{E}_\pi \left[ \sum_{t=0}^{\infty} \gamma^t R_{t+1} \mid S_0 = s \right]"
/>

Expanding one step gives the Bellman expectation equation:

<Math
  proof
  latex={`V^\\pi(s) &= \\mathbb{E}_\\pi \\left[ G_0 \\mid S_0 = s \\right] \\\\
&= \\mathbb{E}_\\pi \\left[ R_1 + \\gamma G_1 \\mid S_0 = s \\right] \\\\
&= \\sum_{a \\in A} \\pi(a \\mid s) \\sum_{s' \\in S} P(s' \\mid s, a) \\left[ R(s, a, s') + \\gamma V^\\pi(s') \\right]`}
/>
